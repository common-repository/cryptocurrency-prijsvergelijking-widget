== Description ==
 
Eenvoudig cryptocurrency prijzen vergelijken met de widget van [https://cryptoprijzen.com/](https://cryptoprijzen.com/). Implementeer eenvoudig met een shortcode de koop en verkoopprijzen van alle brokers van Nederland. Kies uit tientallen verschillende munten als Bitcoin, Ethereum, Litecoin, Ripple en EOS. Je hebt de keuze om de top 3 goedkoopste aanbieders te tonen of een tabel met alle aanbieders.

Er kan een shortcode gegenereerd worden die overal op de website geplaatst kan worden. Plaats de shortcode in een kolom in een blogpost of in de sidebar.

DEMO: [Cryptocurrency Prijsvergelijking Widget](https://cryptoprijzen.com/gratis-cryptocurrency-prijsvergelijking-widget/)

== Installation ==
 
1. Upload `cryptocurrency-prijsvergelijker.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
